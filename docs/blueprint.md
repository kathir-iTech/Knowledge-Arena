# **App Name**: Cyber Gladiators

## Core Features:

- Role-Based Authentication: Automatically assign user roles (Teacher/Student) based on email domain (@staffs.com for Teachers).
- Quiz Creation (Teacher): Teachers can create custom quizzes with questions, answer options, explanations, and timers.
- Automatic Battle Room Generation: Upon quiz creation, a unique battle room is generated with a shareable code and QR code.
- Battle Room Joining (Student): Students can join battles by entering the provided room code.
- Live Quiz & Scoring: Present timed questions, score based on correctness and speed, and provide immediate feedback with explanations.
- XP System & Leaderboard: Award XP for participation and display a ranked leaderboard of users based on total XP.
- Tab Visibility Check: Automatically remove students from the battle if they switch browser tabs during a quiz, ensuring fair play.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to reflect the cyberpunk theme and create a futuristic atmosphere.
- Background color: Dark grey (#222222) to create contrast and enhance the cyberpunk aesthetic.
- Accent color: Neon purple (#D400FF) to highlight interactive elements and provide a vibrant pop.
- Headline font: 'Space Grotesk' sans-serif for a techy, futuristic feel; body font: 'Inter' sans-serif for readability.
- Use sharp, angular icons inspired by cyberpunk aesthetics.
- Implement a modern, grid-based layout with a persistent sidebar for navigation.
- Incorporate subtle animations for transitions and interactive elements to enhance user engagement.
